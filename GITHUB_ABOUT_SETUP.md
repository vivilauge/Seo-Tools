# 如何在 GitHub 仓库 About 部分添加网站链接

## 📍 设置步骤

### 方法一：通过网页界面设置（推荐）

1. **访问你的 GitHub 仓库**
   - 打开 https://github.com/vivilauge/Seo-Tools

2. **点击设置图标**
   - 在仓库页面右侧，找到 "About" 部分
   - 点击右侧的齿轮图标 ⚙️

3. **填写网站地址**
   - 在弹出的对话框中，找到 "Website" 字段
   - 输入你的网站地址（如果项目只本地运行，可以留空或填写本地地址）
   - 点击 "Save changes"

4. **完成！**
   - 网站链接会显示在仓库右侧的 About 部分
   - 点击链接可以直接访问在线演示

### 方法二：通过仓库设置页面

1. **进入仓库设置**
   - 访问仓库页面
   - 点击顶部的 "Settings" 标签页

2. **找到 General 设置**
   - 在左侧菜单中找到 "General"
   - 滚动到 "Features" 部分

3. **启用并设置网站**
   - 勾选 "Use this template" 下方的选项（如果有）
   - 在 "Website" 字段输入你的网站地址（如果项目只本地运行，可以留空或填写本地地址）
   - 点击页面底部的 "Save changes"

## ✅ 设置完成后的效果

设置完成后，在仓库页面的右侧 About 部分会显示网站链接（如果已设置）。

## 📝 注意事项

- 网站 URL 必须以 `http://` 或 `https://` 开头
- GitHub 会自动验证链接的有效性
- 如果链接无效，GitHub 会显示警告
