# Seo-Tools - SEO工具导航站

一个基于纯前端技术的 SEO 工具集合，提供多个实用的 SEO 工具，无需启动服务器，直接在浏览器中运行。

## 功能导航

### 🌐 网站状态码批量检查
- ✅ 批量检查网站URL的状态码
- ✅ 实时显示检查结果
- ✅ 自动处理不带协议的URL（如：`example.com/path/file.php`）
- ✅ 支持HTTP和HTTPS协议选择
- ✅ 状态码分类显示（2xx绿色、3xx蓝色、4xx黄色、5xx红色）
- ✅ 统计信息显示
- ✅ 进度条显示
- ✅ 支持并发检查

### 🔍 文本对比去重复
- ✅ 固定文本列表保持不变
- ✅ 自动检测新文本列表中与固定文本完全一样的文本
- ✅ 过滤掉与固定文本完全一样的新文本
- ✅ 返回去重复后的新文本列表
- ✅ 统计重复和保留的数量

### 🔗 域名对比去重复
- ✅ 固定域名列表保持不变
- ✅ 自动检测新域名列表中与固定域名完全一样的域名（去掉协议、路径和参数后）
- ✅ 过滤掉与固定域名完全一样的新域名
- ✅ 返回去重复后的新域名列表
- ✅ 统计重复和保留的数量

### 🔍 域名重复检测
- ✅ 去掉域名的协议头、路径和参数
- ✅ 提取纯域名后去重
- ✅ 识别重复域名并显示详情
- ✅ 快速清理URL列表中的重复域名

### 📝 文本去重复
- ✅ 去除文本内容中的重复行
- ✅ 支持多种去重规则
- ✅ 保持原始顺序选项
- ✅ 清理和整理文本数据

## 安装和使用

### 快速开始

1. 下载或克隆项目：
```bash
git clone https://github.com/vivilauge/Seo-Tools.git
cd Seo-Tools
```

2. 直接在浏览器中打开 `index.html` 文件，或使用本地服务器：
```bash
# 使用Python启动简单服务器（推荐）
python3 -m http.server 8080
# 然后访问 http://localhost:8080

# 或者直接双击打开 index.html 文件
```

**重要提示：**
- ✅ **纯前端运行**：无需安装 Node.js，所有功能都在浏览器中运行
- ✅ **离线可用**：下载后可以完全离线使用
- ✅ **跨平台兼容**：支持所有现代浏览器

## 使用说明

### 导航首页
打开 `index.html` 文件进入导航首页，可以看到所有可用的 SEO 工具。

### 网站状态码批量检查
1. 点击导航首页的"网站状态码批量检查"卡片
2. 选择并发数（同时检查的域名数量）
3. 输入URL列表，每行一个，必须包含完整协议：
   - 完整URL：`https://example.com/path/file.php`
   - 完整URL：`http://example.com/api/endpoint`
4. 点击"开始检查"按钮，系统会并发检查URL并实时显示结果
5. 可以随时点击"停止检查"按钮停止检查
6. 点击"清空结果"按钮清空所有结果

### 文本对比去重复
1. 点击导航首页的"文本对比去重复"卡片
2. 输入固定文本列表（每行一个，保持不变）
3. 输入新文本列表（每行一个）
4. 点击"开始对比去重"按钮
5. 系统会自动过滤掉新文本中与固定文本完全一样的文本
6. 查看去重复后的新文本列表和统计信息

### 域名对比去重复
1. 点击导航首页的"域名对比去重复"卡片
2. 输入固定域名列表（每行一个，保持不变）
3. 输入新域名列表（每行一个）
4. 点击"开始对比去重"按钮
5. 系统会自动过滤掉新域名中与固定域名完全一样的域名（去掉协议、路径和参数后）
6. 查看去重复后的新域名列表和统计信息

### 域名重复检测
1. 点击导航首页的"域名重复检测"卡片
2. 输入URL列表，每行一个（支持带参数的URL）
3. 点击"开始检测"按钮
4. 系统会自动去掉协议头、路径和参数，提取纯域名后去重
5. 查看去重后的域名列表和重复域名详情

### 文本去重复
1. 点击导航首页的"文本去重复"卡片
2. 输入文本内容，每行一条
3. 选择去重选项（区分大小写、去除空格、移除空行、保持顺序）
4. 点击"开始去重"按钮
5. 查看去重结果和统计信息

## 注意事项

- 所有工具都是纯前端实现，无需服务器支持
- 网站状态码检查依赖浏览器的CORS支持
- 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）以获得最佳体验
